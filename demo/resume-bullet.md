# Resume Bullet

Built an agentic incident response platform for e-commerce API outages using FastAPI, React, LangGraph-style multi-agent orchestration, runbook RAG, log/metric/deployment analysis, evidence ranking, human-in-the-loop approvals, deterministic evals, and automated postmortem generation across a 9-step incident workflow.

## Shorter Version

Built an agentic incident commander for e-commerce API outages using FastAPI, React, runbook RAG, evidence ranking, human approvals, deterministic evals, and automated postmortems across a 9-step investigation workflow.

## Interview One-Liner

It is an AI incident commander that investigates checkout outages by correlating alerts, logs, metrics, deployments, GitHub commits, and runbooks before recommending an approval-gated mitigation.
