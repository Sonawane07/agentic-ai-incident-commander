# Resume Bullet

Built a deployment-ready agentic incident response platform using LangGraph, FastAPI, React, PostgreSQL/pgvector, Ollama, Docker Compose, Prometheus/Grafana, runbook RAG, human-in-the-loop approvals, deterministic evals, and automated postmortem generation for e-commerce API incidents.

## Shorter Version

Built an agentic incident commander for e-commerce API outages using LangGraph, FastAPI, React, runbook RAG, evidence ranking, human approvals, deterministic evals, and automated postmortems.

## Interview One-Liner

It is an AI incident commander that investigates checkout outages by correlating alerts, logs, metrics, deployments, GitHub commits, and runbooks before recommending an approval-gated mitigation.
